# Cocina Estrellada

Web sobre cocina con diseño cozy, colores pasteles y estética moderna.

## Estructura

```
cocina-estrellada/
├── assets/
│   └── logo.png
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── index.html
└── README.md
```

## Ejecutar

Abre `index.html` en el navegador o usa un servidor local:

```bash
cd cocina-estrellada
python -m http.server 8080
```

Luego visita http://localhost:8080
